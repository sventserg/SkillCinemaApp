# Skillcinema-app

Application based on unofficial kinopoisk API (https://kinopoiskapiunofficial.tech/documentation/api/)

Main features:
- showing lists of premieres, popular movies, best movies etc
- showing detailed information about the selected movie or series
- searching for movies or series by criteria
- saving favorites and interesting movies/series
- creating custom lists
